Pengertian: AJAX adalah teknik dalam pengembangan web yang memungkinkan halaman web untuk berkomunikasi dengan server secara asynchronous (tanpa perlu me-reload halaman). Meskipun namanya menyebut XML, saat ini AJAX lebih sering digunakan dengan format JSON.

Pengertian: jQuery adalah library JavaScript yang dirancang untuk menyederhanakan manipulasi HTML DOM, event handling, animasi, dan terutama AJAX.

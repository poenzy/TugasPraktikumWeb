Canvas

Kode HTML ini menampilkan sebuah artikel berjudul "Canvas", yang berisi elemen canvas dengan ID "canvas", berukuran 400 x 400 piksel, dan memiliki latar belakang abu-abu gelap. Elemen ini berfungsi untuk menggambar grafik dan animasi menggunakan JavaScript, yang dihubungkan melalui file eksternal "js/main.js".
